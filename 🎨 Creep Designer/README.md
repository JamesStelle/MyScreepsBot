# 🎨 Creep Designer

Screeps 爬虫皮肤设计器 - 预览皮肤组合效果的可视化工具

## 📖 简介

Creep Designer 是一个专为 Screeps World 玩家设计的工具，用于预览爬虫皮肤组合后的效果。通过简单的拖拽上传，你可以快速查看多个皮肤图层叠加后的最终外观。

## ✨ 特性

- 🖼️ **可视化预览** - 实时查看皮肤组合效果
- 🎯 **固定尺寸输出** - 统一输出 256×256 像素的设计图
- 🎨 **图层叠加** - 支持多个皮肤图层按顺序叠加
- 💾 **一键下载** - 直接保存设计好的爬虫外观
- 🚀 **无需安装** - 基于浏览器，开箱即用
- 📱 **拖拽上传** - 支持拖拽和多选文件

## 🚀 快速开始

### 方法：直接使用

直接用浏览器打开 `combined_image.html` 文件即可使用

## 📋 使用说明

### 1. 准备皮肤图片

将你的爬虫皮肤图片放入 `CreepColor` 文件夹（可选），或直接通过界面上传。

支持的图片格式：
- PNG（推荐，支持透明）
- JPG / JPEG
- GIF
- BMP
- WEBP

### 2. 上传皮肤

- **拖拽上传**：直接将图片拖到上传区域
- **点击上传**：点击上传区域选择文件
- **多选支持**：可以一次选择多个皮肤文件

### 3. 设计爬虫

1. 上传的图片会按顺序显示在预览区
2. 可以点击 ❌ 删除不需要的图片
3. 点击 **🎨 设计爬虫** 按钮开始合成
4. 画布会显示最终的组合效果

### 4. 下载设计

点击 **💾 下载设计** 按钮，保存为 `creep_design.png` 文件

## 🛠️ 命令行工具

### 查看图片列表

查看 `CreepColor` 文件夹中的皮肤文件：

```bash
node Combination.js list
```

### 生成合成器

生成 HTML 合成器文件：

```bash
node Combination.js generate
```

## 🎯 技术特点

- **纯前端实现** - 无需后端服务器
- **Canvas API** - 使用原生 Canvas 进行图像处理
- **响应式设计** - 适配不同屏幕尺寸
- **零依赖** - 不需要安装任何 npm 包

## 💡 使用技巧

1. **图层顺序**：按上传顺序叠加，后上传的图片会覆盖在上面
2. **透明图片**：使用 PNG 格式可以保留透明效果
3. **预览调整**：可以随时删除图片重新组合
4. **批量测试**：一次上传多个皮肤快速预览效果

## 🔧 开发

### 修改输出尺寸

在 `Combination.js` 中修改 `outputSize` 变量：

```javascript
const outputSize = 256; // 修改为你需要的尺寸
```

### 自定义样式

编辑 HTML 中的 `<style>` 部分来自定义界面样式。
